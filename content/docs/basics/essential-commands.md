## 슬래시(/) 명령어란?

일반적인 대화 대신 `/` 기호로 시작하는 특수 명령어입니다. 더 빠르고 간결하게 작업을 제어할 수 있습니다.

---

## 📚 기본 명령어 5가지

### 1️⃣ `/help` — 도움말 보기

```bash
/help
```

**역할:** 사용 가능한 모든 명령어 목록 확인
- 처음 사용할 때 명령어 기억이 안 나면 입력
- Claude Code가 모든 옵션을 알려줍니다

---

### 2️⃣ `/clear` — 대화 초기화

```bash
/clear
```

**역할:** 이전 대화 내용을 지우고 새로 시작
- 다른 주제로 넘어갈 때 입력하면 좋습니다
- 이전 대화 맥락을 완전히 제거합니다
- 비용 절감 효과도 있습니다

**사용 예시:**
```
> 파이썬 코드 만들어줘
> (여러 번 대화)
> /clear
> 이제 HTML 웹사이트 만들어줄래?
```

---

### 3️⃣ `/status` — 현재 상태 확인

```bash
/status
```

**역할:** 사용 중인 모델과 비용 현황 표시
- 지금까지 얼마나 사용했는지 확인
- 남은 크레딧 확인 (유료 사용자)
- API 사용량 모니터링

---

### 4️⃣ `/compact` — 대화 압축

```bash
/compact
```

**역할:** 대화가 길어질 때 요약해서 비용 절약
- 오래된 대화를 요약으로 변환
- 토큰 사용량 감소
- 대화 맥락 유지하면서 비용 절감

---

### 5️⃣ `Ctrl+C` — 작업 중지

터미널에서 `Ctrl` 키와 `C` 키를 **동시에** 누르기

**역할:** AI가 실행 중인 작업을 즉시 멈춤
- 잘못된 방향으로 가고 있으면 중단
- 파일 생성 중간에 멈추기
- 오류가 발생했을 때 중단

**사용 예시:**
```
> 1000개 파일 만들어줘
(AI가 실행 중...)
Ctrl+C 누르기
작업이 멈춥니다
```

## 🚪 종료하고 다시 시작하기

### 종료 방법

두 가지 방법이 있습니다:

```bash
/exit
```

또는

```bash
Ctrl+D
```

### 다시 시작하려면

터미널에서 다시 입력:

```bash
claude
```

자동으로 이전 로그인이 유지되므로 추가 로그인 필요 없습니다.

## 💡 명령어 사용 팁

### 조합 사용하기

```
> 웹사이트 만들어줘
(여러 번 수정 요청)
> /status   (비용 확인)
> /compact  (대화 압축)
> /clear    (새로운 주제 시작)
```

### 언제 사용하면 좋은가?

| 상황 | 명령어 |
|------|--------|
| 명령어 기억 안 남 | `/help` |
| 새로운 주제 시작 | `/clear` |
| 비용 확인하고 싶음 | `/status` |
| 대화가 너무 김 | `/compact` |
| AI 작업이 잘못됨 | `Ctrl+C` |
