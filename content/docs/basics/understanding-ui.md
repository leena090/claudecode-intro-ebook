## 딱 3개 영역만 알면 됩니다

클로드코드의 화면은 3개 영역으로 나뉩니다. 각 영역의 역할을 이해하면 사용이 훨씬 쉬워집니다.

---

## ① 상단 상태 표시

```
Working in: /Desktop/내작업 | claude-opus-4-5 | auto
```

**역할:**
- 현재 작업 중인 **폴더 위치** 표시
- 사용 중인 **AI 모델** 표시
- **권한 모드** 표시 (자동 또는 수동)

**알아야 할 것:**
- 폴더 위치가 맞는지 확인할 때 봅니다
- 모델은 자동으로 선택되므로 신경 쓸 필요 없습니다

---

## ② 대화 내역 영역

```
[나] 안녕하세요, 파이썬 코드 작성 도와줘
[Claude] 네, 어떤 기능을 만들고 싶으신가요?
```

**역할:**
- 나와 Claude의 대화가 **순서대로 표시**
- 이전 대화 내용을 계속 **참고**하면서 작업

**알아야 할 것:**
- 화면을 위로 스크롤하면 이전 대화 확인 가능
- Claude가 이전 대화를 기억하고 문맥을 이해합니다
- 새로운 주제로 시작할 때는 `/clear` 명령어로 초기화하면 좋습니다

---

## ③ 명령 입력창 (프롬프트)

```
> 여기에 요청사항을 입력하세요... █
```

**역할:**
- 클로드코드에게 **요청을 입력**하는 곳
- Enter 키를 눌러 전송

**알아야 할 것:**
- 한국어로 편하게 입력하세요
- 길게 설명해도 괜찮습니다
- 여러 줄로 입력하려면 Shift+Enter를 사용

## 실전 예시

### 상황: 파이썬 코드 작성 요청

```
① 상단에 표시됨
Working in: /Desktop/프로젝트 | claude-opus-4-5 | auto

② 대화 내역
[나] 파이썬으로 간단한 계산기를 만들어줄 수 있을까요?
[Claude] 물론이죠! 어떤 기능을 포함하고 싶으신가요?
[나] 더하기, 빼기, 곱하기 3가지만 되면 좋겠어요
[Claude] 알겠습니다. calculator.py 파일을 만들겠습니다
✓ calculator.py 생성 완료

③ 입력창
> █
```

---

## 각 영역의 크기 조절

터미널의 크기를 늘리면 각 영역도 함께 커집니다:
- 대화가 많으면 → 터미널 창을 크게 열기
- 입력이 길면 → 세로를 길게 조절
