## 예시 대화: 파일 만들기

이렇게 말하면 AI가 바로 실행합니다.

### 내 요청

```
안녕하세요! "hello.txt"라는 파일을 만들고
"안녕하세요, 클로드코드!"라고 내용을 써줄 수 있을까요?
```

### Claude의 응답

```
✓ hello.txt 파일을 생성했습니다

📄 hello.txt
안녕하세요, 클로드코드!

작업 완료! 현재 폴더에서 파일을 확인하실 수 있습니다.
```

**결과:** 현재 폴더에 실제 파일 생성! ✨

---

## 이런 것들을 요청해보세요

### 📝 파일 & 폴더 작업

```
"현재 폴더의 파일 목록 알려줘"
"index.html 파일 만들어줘"
"images 폴더 만들어줘"
```

### 🔍 파일 내용 확인

```
"readme.txt 내용 보여줘"
"이 코드가 뭐 하는 건지 설명해줘"
"오류가 있는지 확인해줘"
```

### ✏️ 내용 수정

```
"이 파일의 제목을 '안녕하세요'로 바꿔줘"
"영어로 번역해줘"
"맞춤법 오류 고쳐줘"
```

---

## 생성할 수 있는 파일의 종류

### 🌐 웹페이지 (HTML/CSS)

```
"나의 포트폴리오 웹사이트를 만들어줘"
"파란색 배경의 랜딩페이지를 만들어줘"
```

### 📊 데이터 파일 (CSV/JSON)

```
"상품 10개의 정보를 CSV로 정리해줘"
"설정 파일을 JSON으로 만들어줘"
```

### 📝 문서 파일 (Markdown/TXT)

```
"프로젝트 README를 작성해줘"
"회의록 템플릿을 만들어줘"
```

### ⚙️ 자동화 스크립트 (Python)

```
"폴더의 모든 파일을 정렬하는 스크립트 만들어줘"
"엑셀 데이터를 처리하는 프로그램 만들어줘"
```

---

## 요청할 때의 팁

### 🎯 구체적으로 요청하기

| ❌ 잘못된 요청 | ✅ 좋은 요청 |
|------------|---------|
| "파일 만들어줘" | "blog.html을 만드는데, 파란 배경에 흰 글씨로, 제목과 3개의 포스트 섹션이 있게 해줘" |
| "코드 짜줘" | "파이썬으로 csv 파일을 읽고 평균을 계산하는 스크립트를 만들어줘" |

### 🗣️ 한국어로 편하게

- 영어 필요 없습니다
- 완전한 한국어로 대화해도 완벽히 이해합니다
- 말투는 존댓말, 반말 모두 가능합니다

### 🔄 수정 요청은 계속 가능

```
> 블로그 웹사이트를 만들어줘
✓ blog.html 생성 완료

> 색을 빨간색으로 바꿔줘
✓ 수정 완료

> 더 크게 만들어줘
✓ 수정 완료
```
