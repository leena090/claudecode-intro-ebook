## 음성 입력 & Fast 모드란?

**음성 입력**은 마이크로 말하기만 하면 AI가 이해하고 작업하는 기능입니다. **Fast 모드**는 같은 AI의 능력을 유지하면서 응답이 더 빨라지는 모드입니다. 마치 자동차의 터보 기능처럼, 더 빠르게 달릴 수 있습니다.

---

## 음성 입력 사용법

### 음성 입력 활성화

```bash
/voice
```

또는

```bash
claude --voice
```

**결과:**
```
🎤 음성 모드 활성화됨
마이크를 확인하세요 (말하면 녹음됩니다)

> 말해주세요: "파일 만들어줄래?"
```

---

## 음성 입력 예시

### 예시 1: 코드 작성

```
당신 (말로): "파이썬으로 간단한 계산기 만들어줄래?"

AI (이해):
> Python 계산기 코드 작성 중...
> 완성!
```

---

### 예시 2: 파일 설명

```
당신 (말로): "이 파일이 뭐 하는 거야?"

AI:
> calc.py 분석 중...
> 계산 함수를 정의하고, 사용자 입력을 받아
> 덧셈, 뺄셈, 곱셈, 나눗셈을 수행합니다.
```

---

### 예시 3: 오류 수정

```
당신 (말로): "이 코드에서 버그를 찾아줄래?"

AI:
> 코드 분석 중...
> 5번째 줄에 오류 발견
> list index out of range
> 수정 완료!
```

## Fast 모드 사용법

### Fast 모드 활성화

```bash
/fast
```

또는

```bash
claude --fast
```

---

### Fast 모드 vs 일반 모드

| 항목 | 일반 모드 | Fast 모드 |
|------|---------|---------|
| 응답 속도 | 보통 | ⚡ 매우 빠름 |
| AI 능력 | 최고 | 95% 수준 |
| 비용 | 기준 | 약간 낮음 |
| 용도 | 복잡한 작업 | 간단한 작업 |

---

## 실전 사용 예시

### 예시 1: 음성 + Fast 모드 조합

```
1️⃣ 음성 활성화
당신: /voice

2️⃣ Fast 모드 활성화
당신: /fast

3️⃣ 음성으로 지시
당신 (말로): "이 파일 요약해줄래?"

AI (빠르게 응답):
> 파일 요약: 사용자 관리 모듈입니다...
> (3초 만에 응답 완료)
```

---

### 예시 2: 회의 중 음성 사용

```
상황: 팀 회의 중
당신 (말로): "이 코드 리뷰해줄래?"

AI (마이크로 듣고):
> 코드 리뷰 완료
> 2가지 개선사항:
> 1. 변수명을 더 명확하게
> 2. 주석 추가 필요
```

---

### 예시 3: 운전 중 지시

```
상황: 자동차 운전 중

당신 (말로): "우리 프로젝트의 진행상황 알려줄래?"

AI:
> 프로젝트 진행율: 65%
> 남은 작업: 35개 중 12개
> 예상 완료: 3주
```

## 음성 입력 설정

### 언어 설정

```bash
/voice --language korean
```

또는 settings.json에서:

```json
{
  "voice": {
    "language": "ko-KR",
    "audioInput": "microphone"
  }
}
```

---

### 마이크 설정

```json
{
  "voice": {
    "device": "Built-in Microphone",
    "sampleRate": 48000,
    "channels": 1
  }
}
```

## Fast 모드 활용법

### Fast 모드에 적합한 작업

```
✅ 간단한 파일 생성
✅ 코드 설명
✅ 빠른 검색
✅ 간단한 수정
✅ 요약
```

---

### Fast 모드 부적합한 작업

```
❌ 복잡한 알고리즘 구현
❌ 깊이 있는 분석
❌ 생각이 많이 필요한 작업
❌ 코드 아키텍처 설계
```

---

## 응답 품질 조절

### /effort 명령어

```bash
/effort low
```

→ 빠르지만 품질은 70% 수준

```bash
/effort medium
```

→ 균형잡힌 속도와 품질

```bash
/effort high
```

→ 최고 품질 (시간 더 걸림)


## 💡 쉽게 이해하기

음성 + Fast는 **택배 배송 옵션**과 같습니다.

**일반 배송:**
- 7~10일 걸림
- 정확하고 꼼꼼함
- 높은 비용

**특급 배송:**
- 하루 만에 도착
- 빠르지만 기본 서비스는 동일
- 약간 더 저렴

마찬가지로:
- **일반 모드:** 정확하고 깊이 있음 (시간 걸림)
- **Fast 모드:** 빠르지만 품질은 여전히 높음
