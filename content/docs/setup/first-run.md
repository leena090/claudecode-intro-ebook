## 첫 실행 3단계

### Step 1: 작업 폴더로 이동

클로드코드가 작업할 폴더로 이동해야 합니다. 예를 들어 바탕화면의 "내작업" 폴더를 사용하려면:

```bash
cd Desktop/내작업
```

또는 홈 폴더에 "projects" 폴더가 있다면:

```bash
cd ~/projects
```

### Step 2: 클로드코드 실행

```bash
claude
```

첫 실행일 때는 로그인 화면이 나타납니다:

```
? How would you like to authenticate?
```

### Step 3: 로그인 방법 선택

#### 추천: Claude.ai 계정으로 로그인

1. 화면에 나타나는 링크 클릭 또는 자동으로 브라우저 열림
2. claude.ai에 로그인 (이미 계정이 있으면)
3. "승인" 또는 "Allow" 버튼 클릭
4. 자동으로 로그인 완료!

#### 대안: API Key로 로그인

1. claude.ai의 설정에서 API Key 생성
2. API Key를 복사
3. 터미널에서 API Key 입력
4. 로그인 완료

## 💡 팁: 로그인 한 번만 하면 됩니다

- 로그인은 **처음 1회만** 필요
- 이후에는 `claude` 명령어만 입력하면 바로 시작
- 자동으로 로그인 상태 유지

---

## 로그인 안 된 경우

### 브라우저가 안 열렸을 때

터미널에 나타나는 URL을 직접 복사해서 브라우저에 붙여넣으세요.

### 이미 로그인되어 있다면

아무것도 안 해도 됩니다. 터미널의 `>` 프롬프트에서 바로 명령을 입력하면 됩니다.

### "인증 실패" 오류

1. 로그아웃 후 다시 시도:
```bash
/exit
claude
```

2. 또는 API Key로 다시 로그인 시도

## 첫 명령 해보기

로그인이 완료되면, 이렇게 입력해보세요:

```bash
> hello world라고 적힌 파일을 만들어줘
```

Claude Code가 자동으로 파일을 생성합니다!
