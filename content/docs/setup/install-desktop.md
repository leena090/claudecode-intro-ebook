## Claude Code는 3가지 방식으로 사용할 수 있습니다

터미널 명령어를 이용한 설치 방식 대신 **데스크톱 앱이나 웹 브라우저**를 통해 더 쉽게 사용할 수 있습니다.

---

## 📱 방식 비교

| 방식 | 특징 | 설치 난이도 | 대상 |
|------|------|----------|------|
| **Desktop 앱** | 설치해서 아이콘으로 실행 | ⭐ 매우 쉬움 | 초보자 추천 |
| **웹 브라우저** | 브라우저에서 바로 사용 | ⭐ 매우 쉬움 | 최고 추천 |
| **터미널 명령어** | npm으로 설치 | ⭐⭐⭐ 어려움 | 개발자 |

---

## 🖥️ Desktop 앱 설치하기

### macOS에서 설치

#### 방법 1: App Store (권장)

1. **App Store 열기**
   - Mac의 Dock에서 App Store 아이콘 클릭
   - 또는 Spotlight (Cmd+Space)에서 "App Store" 입력

2. **Claude Code 검색**
   - 상단 검색창에 "Claude Code" 입력
   - 또는 "Claude" 입력

3. **설치 클릭**
   - "설치" 또는 "가져오기" 버튼 클릭
   - Apple ID 로그인 (필요시)
   - 자동 다운로드 및 설치

4. **실행**
   - 설치 완료 후 "열기" 버튼 클릭
   - 또는 Launchpad에서 "Claude Code" 찾아서 클릭

#### 방법 2: 직접 다운로드

1. **공식 웹사이트 방문**
   - https://claude.ai/download 접속

2. **macOS 버전 다운로드**
   - "Download for macOS" 클릭
   - 자동으로 DMG 파일 다운로드

3. **설치하기**
   - 다운로드된 DMG 파일 더블클릭
   - "Claude Code.app" 아이콘을 Applications 폴더로 드래그
   - 또는 설치 안내 따라하기

4. **실행**
   - Applications 폴더에서 "Claude Code" 더블클릭
   - Dock에 고정하면 다음에 쉽게 클릭 가능

---

### Windows에서 설치

#### 방법 1: Microsoft Store (권장)

1. **Microsoft Store 열기**
   - Windows 시작 메뉴에서 "Microsoft Store" 검색
   - 또는 Windows 키 → "store" 입력

2. **Claude Code 검색**
   - Store 상단 검색창에 "Claude Code" 입력

3. **설치 클릭**
   - "설치" 또는 "가져오기" 버튼 클릭
   - 자동으로 다운로드 및 설치됨

4. **실행**
   - 설치 완료 후 "실행" 클릭
   - 또는 시작 메뉴에서 "Claude Code" 검색해서 실행

#### 방법 2: 직접 다운로드

1. **공식 웹사이트 방문**
   - https://claude.ai/download 접속

2. **Windows 버전 다운로드**
   - "Download for Windows" 클릭
   - EXE 파일 자동 다운로드

3. **설치하기**
   - 다운로드된 설치 파일 더블클릭
   - "설치" 또는 "다음" 버튼 클릭
   - 설치 완료 대기

4. **실행**
   - 설치 완료 후 "실행" 또는 "완료" 클릭
   - 또는 시작 메뉴에서 "Claude Code" 검색

## 🌐 웹 브라우저로 사용하기 (가장 쉬움!)

가장 편한 방법은 **브라우저에서 바로 접속**하는 것입니다. 어떤 컴퓨터에서든 설치 없이 사용할 수 있습니다.

### 사용 방법

1. **브라우저 열기**
   - Chrome, Safari, Firefox, Edge 등 아무 브라우저 사용 가능

2. **주소창에 입력**
   ```
   https://claude.ai/code
   ```

3. **로그인**
   - Claude 계정으로 로그인
   - 없으면 가입 후 로그인

4. **바로 사용**
   - 로그인하면 바로 Claude Code 사용 가능
   - 설치 없음!

### 웹 버전의 장점

✅ **설치 불필요** — 어느 컴퓨터든 바로 사용
✅ **항상 최신** — 자동으로 최신 버전 사용
✅ **크로스 플랫폼** — Windows, Mac, Linux, 심지어 태블릿도 가능
✅ **클라우드 동기화** — 저장된 파일이 자동 백업됨

### 웹 버전의 제한사항

⚠️ **인터넷 필수** — 항상 온라인 상태 필요
⚠️ **기능 일부 제한** — 일부 로컬 기능은 Desktop 앱이 더 나음

## 🔄 Desktop 앱 vs 웹 버전 비교

| 항목 | Desktop 앱 | 웹 버전 |
|------|----------|--------|
| 설치 | 필요 | 불필요 |
| 속도 | 더 빠름 | 약간 느릴 수 있음 |
| 기능 | 모든 기능 | 대부분의 기능 |
| 저장 | 로컬 저장 | 클라우드 저장 |
| 오프라인 | 불가 | 불가 (인터넷 필수) |
| 초보자 추천 | ⭐⭐ | ⭐⭐⭐ |

## 🎯 어떤 방식을 선택할까요?

### Desktop 앱을 추천하는 경우

- 매일 자주 사용하는 경우
- 조금 더 빠른 속도를 원하는 경우
- 로컬에 파일을 저장하고 싶은 경우
- 오프라인에서도 에디터를 보고 싶은 경우

### 웹 버전을 추천하는 경우 (가장 추천!)

- 처음 사용해보는 경우
- 여러 컴퓨터를 사용하는 경우
- 설치 없이 빠르게 시작하고 싶은 경우
- 학교나 회사 PC처럼 설치 권한이 없는 경우
- 어디서나 같은 환경에서 사용하고 싶은 경우

## 💡 쉽게 이해하기

### Desktop 앱 = 스마트폰 앱

- **스마트폰 앱**: App Store에서 설치 → 아이콘 클릭으로 실행
- **Desktop 앱**: App Store/Microsoft Store에서 설치 → 아이콘 클릭으로 실행
- 마찬가지로 편하고 빠릅니다!

### 웹 버전 = 웹브라우저 열기

- **웹사이트**: 주소 입력 → 바로 접속 (예: Google, YouTube)
- **Claude Code 웹**: https://claude.ai/code 입력 → 바로 사용
- 설치 없이 가장 간단합니다!

## 🆘 자주 나오는 문제

### Q: Desktop 앱과 웹 버전에서 파일이 공유되나요?

**A:** 부분적으로 공유됩니다.
- 웹 버전: Anthropic 클라우드에 저장
- Desktop 앱: 로컬 컴퓨터에 저장
- 둘 다 Claude 계정으로 로그인하면 계정 정보는 동기화됨
- 하지만 파일은 따로 관리되므로 각각 열어서 확인 필요

### Q: 두 버전을 동시에 설치할 수 있나요?

**A:** 네, 가능합니다!
- Desktop 앱 설치 + 웹 브라우저 동시 사용 가능
- 상황에 따라 편한 방식 선택

### Q: Desktop 앱은 다운로드 받으면 용량이 얼마나 되나요?

**A:** 약 **300~500MB** 수준입니다.
- 일반적인 SSD라면 충분히 설치 가능
- 설치 시간: 2~5분
