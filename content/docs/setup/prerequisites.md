## 3가지만 확인하면 준비 완료!

### 💻 1. 컴퓨터 (Windows / Mac)

- **필수:** Windows 또는 Mac 운영체제
- **노트북도 OK** — 데스크톱이나 노트북 모두 가능
- **태블릿은 불가** — 아이패드, 안드로이드 태블릿은 사용 불가

---

### 🌐 2. 인터넷 연결

- **다운로드 필요:** 초기 설치 시 파일 다운로드
- **AI 통신 필수:** Claude AI 모델이 Anthropic 서버에서 동작하므로 인터넷 연결 필수

---

### 🔑 3. Anthropic 계정 (필수)

- **무료 가입 가능:** claude.ai에서 무료로 가입 가능
- **API 키 필요:** 클로드코드 사용 시 API 키 필요
- **설치 중 로그인:** 설치 후 처음 실행할 때 로그인

---

## 💰 비용 안내

### 계정 가입

🆓 **Claude 계정 무료 가입 가능**
- claude.ai에서 아무 비용 없이 가입 가능

### 사용 요금

💳 **Claude Code는 API 사용량만큼 과금**
- 문자 개수, 작업 복잡도에 따라 요금이 결정됨
- 일반적으로 월 **$10~20** 수준

### Max Plan

📊 **무제한 사용 플랜도 있습니다**
- 월 **$100**에 무제한 사용 가능
- 헤비 유저(매일 많이 사용)에게는 경제적

---

## 준비 체크리스트

- [ ] Windows 또는 Mac 컴퓨터 확인
- [ ] 인터넷 연결 확인
- [ ] claude.ai에서 계정 가입 (또는 기존 계정 확인)
- [ ] 비용 확인 및 결제 수단 준비

모두 준비되셨으면, 이제 설치를 시작해봅시다!
