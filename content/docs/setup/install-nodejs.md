## Node.js가 뭔가요?

JavaScript 실행 환경입니다. 클로드코드가 동작하기 위해 필요한 **기반 프로그램**으로, 설치 후 특별히 쓸 일은 없습니다. 마치 앱이 실행되려면 Windows가 필요한 것처럼, 클로드코드가 실행되려면 Node.js가 필요한 거죠.

---

## 📥 Node.js 설치 방법

### Step 1: 웹사이트 방문

1. https://nodejs.org 접속
2. **LTS 버전** 다운로드 버튼 클릭
   - LTS = 오래 지원되는 안정적인 버전
   - Current는 최신 기능 버전 (보통 LTS 추천)

### Step 2: 설치 파일 실행

1. 다운로드된 설치파일 실행
2. "Next" 버튼만 계속 클릭
3. "Install" 클릭하면 자동 설치

### Step 3: 설치 확인

터미널/명령 프롬프트에서 다음 명령어를 입력하세요:

```bash
node --version
```

결과 예시:
```
v22.14.0
```

**버전 숫자가 나오면 성공!** 🎉

---

## 터미널이 뭔가요?

명령어를 입력하는 검은 창입니다.

### Windows에서 열기
1. 화면 왼쪽 아래 **검색 창** 클릭
2. **"cmd"** 입력
3. **명령 프롬프트** 실행

### Mac에서 열기
1. 화면 오른쪽 위 **Spotlight** (돋보기 아이콘) 클릭
2. **"터미널"** 입력
3. **터미널** 실행

---

## 이미 설치됐는지 확인

혹시 이전에 이미 설치했을 수도 있습니다. 확인해보세요:

```bash
node -v
```

- **버전 숫자가 나오면**: 이미 설치됨 ✅
  - 그대로 진행해도 괜찮습니다
- **"명령어를 찾을 수 없음" 오류가 나오면**: 새로 설치 필요

## 더 알아보기

### Node.js LTS vs Current
- **LTS (Long Term Support)**: 3년 이상 지원되는 안정 버전
- **Current**: 최신 기능 버전, 6개월마다 변경

클로드코드 초보자는 **LTS 버전**을 권장합니다.

### Mac M1/M2 사용자
- "M1/M2 Native" 버전이 있으면 그것을 선택
- 없으면 기본 버전 선택해도 문제없습니다
