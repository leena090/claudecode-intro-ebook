## Claude Code란?

Claude Code는 Anthropic이 만든 AI 코딩 어시스턴트입니다. 터미널에서 직접 실행하며, 코드 작성부터 디버깅, 파일 관리까지 다양한 작업을 도와줍니다.

## 왜 Claude Code인가?

기존 AI 도구와 달리 Claude Code는:

- **터미널 기반** — 브라우저 없이 명령줄에서 바로 사용
- **파일 시스템 접근** — 프로젝트 파일을 직접 읽고 수정
- **컨텍스트 유지** — 대화 흐름을 기억해 연속적인 작업 가능

## 비개발자도 사용할 수 있나요?

네, 가능합니다. 이 가이드는 비개발자를 위해 설계되었습니다. 코딩 지식 없이도 Claude Code를 활용해 업무를 자동화할 수 있습니다.
