## 슬래시 커맨드가 뭐예요?

Claude Code에서 `/` 기호로 시작하는 명령어입니다. 일반 대화 대신 특정 기능을 빠르게 실행할 때 사용합니다.

**일반 대화:**
```
> 파일을 보여줘
> 코드를 수정해줘
> 상태를 알려줘
```

**슬래시 커맨드:**
```
/help
/status
/clear
```

더 간단하고 빠르죠?

---

## 왜 슬래시 커맨드를 사용하나요?

### 1️⃣ **더 빠르다**
- 문장을 안 써도 됨
- 원하는 기능을 즉시 실행
- 시간 절약

### 2️⃣ **명확하다**
- "지금 뭘 하고 싶은가"가 명확
- 오해의 여지 없음
- 정확한 결과 얻음

### 3️⃣ **비용 절감**
- `/compact`로 대화 압축 → 토큰 사용량 줄음
- `/clear`로 불필요한 맥락 제거
- 효율적인 사용

---

## 슬래시 커맨드의 종류 (4가지 카테고리)

<div class="note-circle">
○ 슬래시 커맨드는 4가지 종류로 나뉩니다
</div>

### 📋 **세션 관리** (대화 제어)
- `/clear` — 대화 초기화
- `/resume` — 이전 대화 재개
- `/rename` — 세션 이름 변경
- `/branch` — 대화 분기
- `/rewind` — 이전 상태로 돌아가기
- `/exit` — 종료

### ⚙️ **설정 관리** (Claude Code 커스터마이징)
- `/config` — 설정 조회/변경
- `/model` — AI 모델 선택
- `/permissions` — 권한 설정
- `/theme` — 화면 테마
- `/color` — 색상 설정
- `/language` — 언어 설정

### ℹ️ **정보 확인** (현황 파악)
- `/status` — 사용 현황
- `/cost` — 비용 내역
- `/usage` — 사용량
- `/stats` — 통계
- `/context` — 컨텍스트 정보
- `/doctor` — 진단
- `/help` — 도움말

### 📁 **파일 & 코드** (작업 도구)
- `/diff` — 변경사항 비교
- `/copy` — 코드 복사
- `/export` — 내보내기
- `/compact` — 대화 압축
- `/memory` — 기억 관리

### 🎯 **특수 기능** (고급 기능)
- `/init` — 새 프로젝트 초기화
- `/plan` — 계획 수립
- `/fast` — 빠른 모드
- `/voice` — 음성 모드
- `/effort` — 작업량 조절
- `/sandbox` — 격리 환경
- `/pr` — PR 생성
- `/review` — 코드 리뷰
- `/commit` — 커밋 메시지

---

## 🚀 슬래시 커맨드 사용하는 법 (3단계)

### Step 1️⃣ — 슬래시 입력
터미널 또는 입력창에 `/` 입력

### Step 2️⃣ — 명령어 선택
**자동완성(Tab)** 으로 선택하거나 전체 명령어 입력
```bash
/sta
(Tab 키 누르면) → /status 자동 완성
```

### Step 3️⃣ — 엔터(Enter) 누르기
```bash
/status
(Enter)
```

실행됩니다!

<mark>Shift+Tab으로 자동완성 목록을 역순으로 이동할 수 있습니다</mark>

## ★ 가장 자주 사용하는 명령어 TOP 5

<div class="note-star">
★ 초보자가 꼭 알아야 할 명령어 5가지
</div>

| 순위 | 명령어 | 용도 | 사용 시점 |
|------|--------|------|----------|
| 1️⃣ | `/help` | 모든 명령어 목록 보기 | 명령어 기억 안 날 때 |
| 2️⃣ | `/status` | 비용 & 사용량 확인 | 요금이 궁금할 때 |
| 3️⃣ | `/clear` | 대화 초기화 | 새로운 주제 시작할 때 |
| 4️⃣ | `/exit` | 세션 종료 | 작업 끝났을 때 |
| 5️⃣ | `/compact` | 대화 압축 | 대화가 너무 길 때 |

## 💡 쉽게 이해하기

### 일상 생활로 비유하면?

슬래시 커맨드는 **리모컨의 버튼**과 같습니다.

- **일반 대화** = "TV를 켜줄 수 있을까?" (길고 번거로움)
- **슬래시 커맨드** = TV 리모컨의 전원 버튼 (한 번에 끝)

원하는 기능을 버튼처럼 빠르게 실행할 수 있다는 뜻입니다.

### 예를 들어...

당신이 지금까지 Claude Code와 3시간을 대화했다고 합시다:
- "컨텍스트가 너무 길어졌어. 새로운 주제로 시작하고 싶은데..."
- **해결책:** `/clear` 한 번만 입력하면 끝!

또는 요금이 얼마나 사용됐는지 궁금할 때:
- "지금까지 얼마나 썼을까?" (일일이 계산해야 함)
- **해결책:** `/cost` 또는 `/status` 입력하면 한눈에 볼 수 있음!
