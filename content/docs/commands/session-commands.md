## 세션이란?

Claude Code와 나누는 **한 번의 연속된 대화**를 의미합니다. 처음 Claude Code를 시작해서 종료할 때까지가 하나의 세션입니다.

세션 관리 명령어는 이 대화를 더 잘 제어하고 정리하는 도구입니다.

---

## 세션 관리 명령어 6가지

### 1️⃣ `/clear` — 대화 모두 지우기

```bash
/clear
```

**역할:** 이전 모든 대화를 삭제하고 새로운 대화 시작

<div class="note-circle">
○ 한 세션 안에서 여러 주제를 다룰 때 특히 유용합니다
</div>

**사용 예시:**
```
> Python 프로그래밍을 배우고 싶어요
> (파이썬 대화 10번)
> /clear
> 이제 HTML 웹사이트 만들어줄래?
(파이썬 대화는 완전히 잊음)
```

**장점:**
- 토큰 비용 절감 (이전 대화를 메모리에 유지 안 함)
- 새로운 주제에 집중
- 혼란스러운 맥락 제거

**주의:**
- `/clear` 후 이전 대화 내용을 복구할 수 없습니다
- 중요한 결과물은 미리 저장해두세요

---

### 2️⃣ `/resume` — 이전 대화 재개하기

```bash
/resume
```

**역할:** `/clear` 후에 이전 대화 맥락을 다시 불러오기

<mark>새로운 세션(Claude를 다시 시작한 후)에 사용합니다</mark>

**사용 예시:**
```
[세션 1]
> 파이썬 함수 설명해줘
> (여러 번 대화)
> /exit

[세션 2]
> /resume
(세션 1의 대화가 다시 불러와집니다)
```

**장점:**
- 여러 세션에 걸쳐 일관된 맥락 유지
- 중단했던 작업 다시 시작 가능

---

### 3️⃣ `/rename` — 세션 이름 바꾸기

```bash
/rename "새로운 이름"
```

**역할:** 현재 세션에 의미 있는 이름 지정

**사용 예시:**
```bash
/rename "파이썬 프로젝트 - 2026년 3월"
/rename "웹사이트 개발 v2"
/rename "버그 수정 모음"
```

**장점:**
- 나중에 세션 목록에서 어떤 작업을 했는지 한눈에 파악
- 중요한 프로젝트 구분 가능
- 협업할 때 다른 사람도 이해하기 쉬움

---

### 4️⃣ `/branch` — 대화 분기하기

```bash
/branch "분기 이름"
```

**역할:** 현재 대화의 갈래를 만들어서 두 개의 다른 방향으로 진행

<div class="note-circle">
○ 마치 게임에서 세이브 포인트를 만들고 다른 엔딩을 보는 것처럼 작동합니다
</div>

**사용 예시:**
```
> 웹사이트 디자인 만들어줘
> (디자인 A 방향으로 진행...)
> /branch "디자인 B 시도"
(이제 다른 디자인 방향으로 테스트 가능)
```

**장점:**
- 여러 가능성을 동시에 탐색
- 원본 대화는 유지됨
- 나중에 어느 쪽 결과가 더 좋은지 비교 가능

---

### 5️⃣ `/rewind` — 이전 상태로 돌아가기

```bash
/rewind
```

또는 특정 시점으로:

```bash
/rewind 3
(3 턴(대화) 전으로 돌아감)
```

**역할:** 실수한 대화를 되돌리거나 다른 방향으로 다시 시작

<mark>마치 게임의 '로드 게임(Ctrl+Z 언두)' 같습니다</mark>

**사용 예시:**
```
> 파이썬 코드 만들어줘
> (AI가 코드 제시)
> 이건 좀 복잡해 보이는데...
> /rewind
(이전 상태로 돌아감)
> 더 간단하게 만들어줄래?
```

**장점:**
- 잘못된 방향 수정
- 다시 입력할 필요 없음
- 효율적인 대화 진행

**주의:**
- `/rewind` 후 그 이후 모든 대화는 사라집니다

---

### 6️⃣ `/exit` — 세션 종료하기

```bash
/exit
```

또는 키보드:

```bash
Ctrl+D
```

**역할:** Claude Code를 종료하고 세션 저장

<div class="note-star">
★ 중요: `/exit`를 사용할 때까지 대화 내용이 저장되지 않을 수 있습니다
</div>

**사용 예시:**
```
> 파이썬 학습 (여러 번 대화)
> 오늘 학습 끝!
> /exit
(세션이 저장되고 Claude Code가 닫힘)
```

**다시 시작하려면:**
```bash
claude
```

이전 로그인이 유지되므로 추가 로그인 필요 없습니다.

## 🔄 세션 명령어 조합하기

실제 사용 예시:

```bash
# 1. 새로운 주제로 시작
/rename "HTML 웹사이트 만들기"

# 2. 처음 디자인 시도
> 상단 메뉴 만들어줘
> (코드 받음)

# 3. 다른 방향도 보고 싶음
/branch "다크테마 시도"
> 다크테마로 바꿀 수 있을까?

# 4. 아, 이건 아니네...
/rewind

# 5. 원래 계획으로 돌아감
> 아, 맞다. 원래대로 계속 진행해줄래?

# 6. 일단 이 정도면 충분
/exit
```

## 💡 쉽게 이해하기

### 세션 관리를 게임으로 비유하면?

Claude Code의 세션 관리 명령어는 **게임의 세이브/로드 기능**과 같습니다:

- **`/clear`** = 새로운 게임 시작 (이전 진행상황 초기화)
- **`/resume`** = 마지막 세이브 파일 불러오기
- **`/rename`** = 세이브 파일 이름 바꾸기 (예: "보스전 직전")
- **`/branch`** = 현재 진행상황 복사해서 다른 엔딩 시도
- **`/rewind`** = 게임 시간을 되감기 (Ctrl+Z)
- **`/exit`** = 게임 종료하고 진행상황 저장

### 예를 들어...

**상황 1: 웹사이트 디자인 고민**
```
> 웹사이트 헤더 디자인해줘
(AI가 디자인 A 제시)
> 음, 다른 스타일도 보고 싶은데...
/branch "스타일 B"
(스타일 B로 계속 진행)
(나중에 둘 중 더 좋은 걸 선택 가능)
```

**상황 2: 실수한 대화 다시 하기**
```
> Python 함수 만들어줘
> (AI가 복잡한 코드 제시)
> 아, 이건 너무 복잡한데?
/rewind
> 더 간단한 예제부터 시작해줄래?
```
