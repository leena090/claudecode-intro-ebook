## 키보드 단축키란?

특정 키 조합을 눌러서 빠르게 기능을 실행하는 방법입니다. 마치 스마트폰의 화면 제스처처럼, 손가락만으로 여러 키보드 입력을 한 번에 처리합니다.

---

## 🎯 가장 중요한 단축키 TOP 5

| 순위 | 단축키 | 작동 | 사용 시점 |
|------|--------|------|----------|
| 1️⃣ | `Enter` | 명령 전송 | 입력 완료했을 때 |
| 2️⃣ | `Ctrl+C` | 작업 중단 | AI가 잘못 실행 중일 때 |
| 3️⃣ | `Ctrl+D` | 세션 종료 | 작업 끝났을 때 |
| 4️⃣ | `Shift+Tab` | 자동완성 (역순) | 명령어 목록 보기 |
| 5️⃣ | `↑ / ↓ 방향키` | 이전/다음 명령 | 이전 입력 반복할 때 |

## 📋 전체 키보드 단축키

### 📝 기본 입력 & 전송

입력과 제출 관련 단축키입니다.

| 단축키 | 작동 | 설명 | 예시 |
|--------|------|------|------|
| `Enter` | 명령 전송 | 입력한 명령을 AI로 전송 | 입력 완료 후 Enter 누르기 |
| `Shift+Enter` | 줄바꿈 | 긴 입력 시 줄을 나누기 | 여러 줄 입력할 때 |
| `Tab` | 자동완성 (정방향) | 슬래시 명령어 자동완성 | `/st` 입력 후 Tab → `/status` |
| `Shift+Tab` | 자동완성 (역순) | 자동완성 목록을 거꾸로 이동 | 이전 옵션 선택할 때 |
| `Escape` | 입력 취소 | 현재 입력 줄 지우기 | 잘못 입력했을 때 |

### ⏸️ 작업 제어

프로그램 실행을 제어하는 단축키입니다.

| 단축키 | 작동 | 설명 | 주의사항 |
|--------|------|------|----------|
| `Ctrl+C` | 작업 중단 | 실행 중인 작업 즉시 멈추기 | 파일 생성 중도 중단 가능 |
| `Ctrl+Z` | 정지 (일시중지) | 작업을 일시중지 (재개: `fg`) | 터미널에서만 사용 가능 |
| `Ctrl+D` | 세션 종료 | Claude Code 완전히 종료 | 데이터 저장됨 |
| `Ctrl+L` | 화면 정리 | 터미널 화면 지우기 | 화면이 복잡할 때 사용 |

### 🔍 히스토리 & 네비게이션

이전 명령을 다시 사용하고 탐색하는 단축키입니다.

| 단축키 | 작동 | 설명 | 팁 |
|--------|------|------|-----|
| `↑ 위 화살표` | 이전 명령 | 이전에 입력한 명령 불러오기 | 같은 명령 반복할 때 |
| `↓ 아래 화살표` | 다음 명령 | 히스토리에서 아래로 이동 | 위로 가서 다시 아래로 |
| `Ctrl+R` | 검색 | 히스토리에서 명령 검색 | `/st` 명령 찾을 때 |
| `Ctrl+A` | 줄 처음으로 | 커서를 줄 처음으로 이동 | 앞부분 수정할 때 |
| `Ctrl+E` | 줄 끝으로 | 커서를 줄 끝으로 이동 | 뒷부분 수정할 때 |

### 📋 편집 & 삭제

텍스트를 편집하고 삭제하는 단축키입니다.

| 단축키 | 작동 | 설명 | 예시 |
|--------|------|------|------|
| `Ctrl+U` | 줄 처음부터 삭제 | 커서 앞 모든 텍스트 삭제 | `/long command` 다 지울 때 |
| `Ctrl+K` | 줄 끝까지 삭제 | 커서 뒤 모든 텍스트 삭제 | 뒷부분만 지울 때 |
| `Ctrl+W` | 단어 삭제 | 이전 단어 삭제 | 마지막 단어만 지우기 |
| `Ctrl+H` 또는 `Backspace` | 한 글자 삭제 | 커서 앞 한 글자 삭제 | 오타 수정 |
| `Ctrl+D` | 커서 위치 글자 삭제 | 커서 위의 글자 삭제 | 중간 글자 삭제 |

### 🎯 슬래시 명령어 관련

슬래시 명령어를 빠르게 입력하는 단축키입니다.

| 단축키 | 작동 | 설명 | 예시 |
|--------|------|------|------|
| `/` + `Tab` | 명령 목록 | 모든 슬래시 명령어 목록 표시 | 뭘 입력해야 할지 모를 때 |
| `/st` + `Tab` | 자동완성 | `/status` 자동 완성 | 빠르게 상태 확인 |
| `/cl` + `Tab` | 자동완성 | `/clear` 자동 완성 | 빠르게 초기화 |
| `/he` + `Tab` | 자동완성 | `/help` 자동 완성 | 도움말 빠르게 보기 |

### 💾 복사 & 붙여넣기

클립보드 작업 관련 단축키입니다.

| 단축키 | 작동 | 설명 | 팁 |
|--------|------|------|-----|
| `Ctrl+C` | 복사 | 선택한 텍스트 복사 | 마우스로 드래그 후 사용 |
| `Ctrl+V` | 붙여넣기 | 클립보드 내용 붙여넣기 | 코드나 텍스트 복사 후 |
| `Ctrl+X` | 자르기 | 선택한 텍스트 잘라내기 | 이동할 때 사용 |

> ⚠️ **주의:** `Ctrl+C`는 복사와 중단 두 가지 역할을 합니다.
> - 텍스트 선택 상태: 복사
> - 명령 입력 상태: 중단

## 🎯 상황별 단축키 조합

### 상황 1: 이전 명령 다시 사용하고 싶을 때

```
↑ 위 화살표
(이전 명령 나타남)
↑ 또는 ↓로 원하는 명령 찾기
Enter
(명령 실행)
```

### 상황 2: 비슷한 명령을 빠르게 여러 번 입력

```
↑ (이전 명령)
Ctrl+A (줄 처음)
오른쪽 화살표로 원하는 위치 이동
필요한 부분 수정
Enter (실행)
```

### 상황 3: 슬래시 명령어가 기억 안 날 때

```
/
Tab (또는 Shift+Tab)
(모든 슬래시 명령어 목록)
↑ 또는 ↓로 선택
Enter
```

### 상황 4: 긴 명령을 여러 줄로 입력

```
첫 번째 라인 입력
Shift+Enter (줄바꿈)
두 번째 라인 입력
Shift+Enter
마지막 라인 입력
Enter (전송)
```

### 상황 5: 입력 실수해서 다 지우고 싶을 때

```
Ctrl+U (줄 처음부터 삭제)
또는
Escape (전체 삭제)
```

## 💡 쉽게 이해하기

### 단축키를 자동차 조작으로 비유하면?

| 단축키 | 자동차 | 설명 |
|--------|--------|------|
| `Enter` | 출발 버튼 누르기 | 명령을 시작 |
| `↑ / ↓` | 기어 변속 | 이전 명령 찾아가기 |
| `Ctrl+C` | 비상정지 버튼 | 즉시 중단 |
| `Tab` | 자동 주차 | 자동 완성 |
| `Ctrl+D` | 시동 끄기 | 완전히 종료 |

### 예를 들어...

**예시 1: 같은 작업 여러 번**
```
첫 번째:
> /status
Enter

두 번째 (3초 후):
↑ (위 화살표)
Enter

세 번째 (3초 후):
↑ (위 화살표)
Enter

→ 매번 치지 않아도 됨!
```

**예시 2: 명령어 이름 까먹었을 때**
```
> /st
(뭐가 있는지 모를 때)
Tab (또는 Shift+Tab)
→ /status, /stats, /start 등이 나타남
↓ 또는 ↑로 선택
Enter
```

**예시 3: 긴 프롬프트 수정**
```
> 파이썬 함수 만들어줄래? 그런데 변수명은 명확하게
(실수함)
Ctrl+A (처음으로)
↑ (오른쪽 화살표) 여러 번
"파이썬" 단어 찾아 수정
Enter
```

## 📊 단축키 체계

### Ctrl 관련
- `Ctrl+C` — 중단/복사
- `Ctrl+D` — 종료
- `Ctrl+A` — 줄 처음
- `Ctrl+E` — 줄 끝
- `Ctrl+U` — 앞 삭제
- `Ctrl+K` — 뒤 삭제
- `Ctrl+W` — 단어 삭제
- `Ctrl+R` — 검색

### Shift 관련
- `Shift+Enter` — 줄바꿈
- `Shift+Tab` — 자동완성 (역순)

### 화살표 관련
- `↑ / ↓` — 히스토리 이동
- 마우스로 드래그 후 방향키 — 선택 확장

### 특수키
- `Tab` — 자동완성
- `Escape` — 취소
- `Enter` — 전송

## 🎯 빠른 참고표

### 가장 자주 쓰는 5가지
```
Enter     - 명령 전송
↑ / ↓    - 이전/다음 명령
Ctrl+C    - 중단
Tab       - 자동완성
Escape    - 취소
```

### 초보자 필수 5가지
```
Enter        - 입력 완료
Ctrl+C       - 잘못됐을 때 중단
Ctrl+D       - 종료
↑ 위 화살표  - 이전 명령 다시 사용
Tab          - 명령어 자동완성
```

### 고급 사용자 팁
```
Ctrl+U      - 줄 처음부터 빠르게 삭제
Ctrl+R      - 히스토리 검색
Shift+Enter - 여러 줄 입력
Ctrl+A/E    - 줄 처음/끝으로 즉시 이동
```

## 🔍 터미널 종류별 단축키

### macOS (Mac 터미널)
- `Ctrl+U` — 줄 앞 삭제 ✓
- `Ctrl+K` — 줄 뒤 삭제 ✓
- `Cmd+C` — 복사 (Ctrl+C가 아님)
- `Cmd+V` — 붙여넣기

### Windows (CMD 또는 PowerShell)
- `Ctrl+C` — 중단/복사 ✓
- `Ctrl+V` — 붙여넣기 ✓
- `Ctrl+Z` — 되돌리기 (일시중지)
- 일부 Linux 단축키는 작동 안 할 수 있음

### Linux (모든 터미널)
- `Ctrl+U` — 줄 앞 삭제 ✓
- `Ctrl+K` — 줄 뒤 삭제 ✓
- `Ctrl+C` — 중단 ✓
- 모든 표준 단축키 지원

## 💡 생산성 팁

### Tip 1: 자주 쓰는 명령 메모
```bash
↑ 위 화살표로 빠르게 접근 가능
최근 명령은 쉽게 반복 가능
```

### Tip 2: 자동완성으로 오타 줄이기
```bash
/st + Tab
→ 정확한 명령어로 자동 완성
오타 가능성 제로
```

### Tip 3: 여러 명령을 한 번에
```bash
첫 번째 명령
Shift+Enter
두 번째 명령
Enter
→ 두 명령이 연속으로 실행
```
