## 빨간 오류 메시지? 그냥 복사-붙여넣기하세요

프로그래밍에서 가장 답답한 부분이 오류입니다. 하지만 클로드코드에게는 쉬운 작업입니다. 오류 메시지를 그냥 복사해서 보여주면 됩니다.

---

## 실전 예시: 오류 해결

### 내 요청

```
코드 실행하니까 이런 오류가 나요. 고쳐줘:

ModuleNotFoundError:
No module named 'requests'
```

### Claude의 응답

```
원인: requests 라이브러리가 설치 안 됨

해결 중...
$ pip install requests

설치 완료! 이제 다시 실행해보세요 ✅
```

**결과:** 오류 해결! 프로그램이 다시 실행됩니다! ✨

---

## 🚨 오류 유형별 대처법

### 🔴 빨간 오류 (Error)

**상황:** 프로그램이 실행되다가 중단되고 빨간 글씨 오류 메시지

**대처법:**
1. 오류 메시지 **전체를 복사**
2. Claude Code에 붙여넣기
3. **"이 오류 고쳐줘"라고 요청**

```
> 이 오류가 나왔어. 고쳐줘:

  TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

---

### 🟡 예상과 다른 결과

**상황:** 프로그램은 실행되는데 결과가 원하는 것이 아님

**대처법:**
1. **원하는 결과** 설명
2. **실제 결과** 설명
3. Claude Code에 요청

```
> 이 프로그램이 1부터 10까지 더한 값을 보여줘야 하는데
  50이 나와요. 뭐가 잘못된 걸까?
```

---

### 🟢 예방적 검토

**상황:** 코드를 실행하기 전에 미리 문제가 있는지 확인하고 싶음

**대처법:**

```
> 이 코드를 실행하기 전에 문제가 있는지 검토해줄 수 있을까?
```

## 다양한 오류 상황

### 1️⃣ 설치 오류

```
> pip install requests 하니까 오류가 나요:

  error: could not create directory '/usr/local/lib/python3.9'
```

→ Claude가 권한 문제를 해결해줍니다.

---

### 2️⃣ 문법 오류

```
> 이 코드에 문법 오류가 있다고 하는데:

  File "app.py", line 5
    print('hello'
          ^
  SyntaxError: '(' was never closed
```

→ Claude가 문법을 수정해줍니다.

---

### 3️⃣ 논리 오류

```
> 이 계산 결과가 이상해요
  5 * 3을 계산하면 15가 나와야 하는데 45가 나와요
```

→ Claude가 논리를 분석하고 수정해줍니다.

## 오류 메시지 효과적으로 전달하기

### ❌ 너무 간단한 설명

```
"오류가 났어요"
"프로그램이 안 돼요"
"뭔가 잘못됐어요"
```

### ✅ 정확한 설명

```
"이 명령어를 실행했을 때 이 오류가 나요:

 [오류 메시지 전체 복사]

 어떻게 해야 할까요?"
```

---

## 오류 메시지를 복사하는 법

### Windows (명령 프롬프트)

1. 오류 텍스트를 마우스로 선택
2. Ctrl+C 누르기
3. Claude Code에 Ctrl+V 붙여넣기

### Mac (터미널)

1. 오류 텍스트를 마우스로 선택
2. Command+C 누르기
3. Claude Code에 Command+V 붙여넣기

---

## 오류를 설명할 때의 팁

### 1. 실행한 명령어 포함

```
> python app.py를 실행했을 때 이 오류가 나요:
  [오류 메시지]
```

### 2. 파일 내용도 함께 보여주기

```
> app.py 파일의 오류를 고쳐줄 수 있을까?
  [파일 내용 또는 코드 일부 붙여넣기]
```

### 3. 설치나 설정 오류는 배경 설명

```
> Windows 컴퓨터에서 Python 설치 후 첫 번째로
  패키지를 설치하려고 하는데 이 오류가 나요:
  [오류 메시지]
```

## 여전히 안 되면?

### 1. 다시 한 번 요청하기

```
> 위의 해결책을 따라 했는데 여전히 오류가 나요.
  이제 이런 오류가 나와요:
  [새로운 오류 메시지]
```

### 2. 설명을 다르게 요청하기

```
> 처음부터 차근차근 설명해줄 수 있을까요?
  오류가 뭐고, 왜 발생했고, 어떻게 해야 하는지
```

### 3. 전문가의 도움 요청

```
> 이 문제를 해결하려면 뭘 공부해야 할까요?
```

## 💡 팁

### 자주 마주치는 오류들

| 오류 | 원인 | 해결 |
|------|------|------|
| ModuleNotFoundError | 라이브러리 설치 안 됨 | `pip install [라이브러리명]` |
| FileNotFoundError | 파일을 찾을 수 없음 | 파일 경로 확인, 파일 생성 |
| TypeError | 데이터 타입 불일치 | 데이터 타입 변환 또는 수정 |
| IndentationError | 들여쓰기 오류 | 들여쓰기 수정 |

### 오류 메시지 활용의 진짜 가치

✅ 그냥 Claude에게 보여주기
