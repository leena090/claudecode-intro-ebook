## 아이디어를 말하면 파일로 만들어줍니다

클로드코드의 가장 기본이면서도 강력한 기능입니다. "이런 파일 필요해"라고 말하면 바로 생성됩니다.

---

## 실전 예시: 블로그 HTML 만들기

### 내 요청

```
"나의 첫 번째 블로그"라는 제목의 간단한 HTML 파일 만들어줘. 소개글도 넣어줘.
```

### Claude의 응답

```
✓ blog.html 생성 완료

<!DOCTYPE html>
<html>
  <h1>나의 첫 번째 블로그</h1>
  <p>안녕하세요...</p>
</html>

작업 완료! 현재 폴더에서 파일을 확인하실 수 있습니다.
```

**결과:** 현재 폴더에 blog.html 파일이 생성되었습니다! ✨

## 만들 수 있는 파일 종류

### 🌐 웹페이지 (HTML/CSS)

- 랜딩페이지 (제품, 서비스 소개)
- 포트폴리오 사이트
- 개인 소개 페이지
- 온라인 매장
- 블로그 페이지

**예시:**
```
"우리 회사의 소개 페이지를 HTML로 만들어줘"
"포트폴리오를 보여주는 웹사이트 만들어줘"
"파란색 배경의 예쁜 랜딩페이지 만들어줘"
```

---

### 📊 데이터 파일 (CSV/JSON)

- 상품 목록 정리
- 고객 데이터 정리
- 설정 파일
- API 응답 예시

**예시:**
```
"10개 상품의 이름, 가격, 설명을 CSV로 만들어줘"
"사용자 정보를 JSON 형식으로 변환해줘"
"설정값을 JSON 파일로 만들어줘"
```

---

### 📝 문서 파일 (Markdown/TXT)

- README 파일 (프로젝트 설명)
- 기획서
- 회의록 템플릿
- 사용 설명서

**예시:**
```
"GitHub용 README 파일을 작성해줘"
"프로젝트 기획서 템플릿 만들어줘"
"회의록 양식을 마크다운으로 만들어줘"
```

---

### ⚙️ 자동화 스크립트 (Python)

- 파일 정리 자동화
- 데이터 처리
- 반복 작업 자동화
- 데이터 변환

**예시:**
```
"폴더의 모든 .txt 파일을 읽어서 한 개의 파일로 합쳐주는 파이썬 스크립트"
"엑셀 파일의 첫 줄을 헤더로 인식하는 CSV 변환 프로그램"
"매일 아침 9시에 이메일을 보내는 자동화 스크립트"
```

## 구체적으로 요청하는 법

### ❌ 너무 추상적인 요청

```
"웹사이트 만들어줘"
"코드 짜줘"
"파일 만들어줘"
```

### ✅ 구체적인 요청

```
"나의 온라인 쇼핑몰을 만들어줘.
상단에 네비게이션 메뉴가 있고,
상품을 그리드로 보여주는 페이지를 원합니다.
색상은 파란색과 흰색으로 해줘."
```

**차이:**
- 구체적인 요청 = 원하는 결과물을 정확히 얻기 가능
- 추상적인 요청 = Claude가 추측해서 만들어야 함

## 생성된 파일 수정하기

### 처음 생성

```
> 계산기 HTML 파일을 만들어줘
✓ calculator.html 생성 완료
```

### 수정 요청 1

```
> 배경색을 검은색으로 바꿔줘
✓ 수정 완료
```

### 수정 요청 2

```
> 버튼을 더 크게 만들어줘
✓ 수정 완료
```

### 수정 요청 3

```
> 글씨는 흰색으로 해줘
✓ 수정 완료
```

## 💡 팁

### 1. 여러 파일 한 번에 만들기

```
"이 HTML에 맞는 CSS 파일과 JavaScript 파일도 만들어줘"
```

### 2. 이미 있는 파일 활용하기

```
"지금 있는 style.css 파일을 보고,
이 HTML에 맞게 수정해줄 수 있을까?"
```

### 3. 템플릿 요청하기

```
"블로그를 위한 HTML 템플릿을 만들어줘.
여러 포스트를 추가할 수 있도록 해줘"
```
