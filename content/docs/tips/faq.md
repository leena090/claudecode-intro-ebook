## 🔗 자주 묻는 질문들

### Q1: ChatGPT랑 다른 게 뭔가요?

**A:** ChatGPT는 대화만 합니다. 클로드코드는 **실제로 내 컴퓨터 파일을 읽고 만들고 수정까지 합니다.** 실행 능력이 핵심 차이입니다.

**비유:**
- ChatGPT = 전화로 조언해주는 전문가
- 클로드코드 = 집에 와서 직접 고쳐주는 기술자

**차이점:**
| 구분 | ChatGPT | 클로드코드 |
|------|---------|----------|
| 대화 | ✅ | ✅ |
| 파일 생성 | ✗ | ✅ |
| 파일 수정 | ✗ | ✅ |
| 코드 실행 | ✗ | ✅ |

---

### Q2: 인터넷이 없으면 못 쓰나요?

**A:** 네, **인터넷 연결이 필수**입니다.

**왜?**
- AI 모델이 Anthropic 서버에서 동작합니다
- 클라우드 연결이 필수입니다
- 매 요청마다 인터넷으로 통신합니다

**필요한 인터넷 속도:**
- 일반적인 가정용 인터넷이면 충분합니다 (5Mbps 이상)
- WiFi도 가능, 유선도 가능

---

### Q3: 내 파일이 외부로 유출되나요?

**A:** 대화 내용은 **Anthropic 서버로 전송**됩니다.

**안전성:**
- Anthropic은 보안에 매우 신경씀
- 데이터 암호화 진행
- 하지만 100% 절대 안전은 아님

**주의사항:**
- ⚠️ **개인정보를 많이 입력하지 마세요**
- ⚠️ **기업 기밀이나 민감한 데이터는 피하세요**
- ✅ 일반적인 작업은 걱정 없습니다

**예시:**
```
❌ 위험: "우리 회사의 고객 명단 정리해줘"
❌ 위험: "개인정보 데이터 변환해줘"

✅ 안전: "샘플 고객 데이터를 정렬해줘"
✅ 안전: "HTML 웹사이트 만들어줘"
```

### Q4: 월 얼마나 나오나요?

**A:** **사용량에 따라 다르지만 일반적으로 월 $5~20** 수준입니다.

**비용 결정 요소:**
- 문자 개수 (입력 + 출력)
- 작업 복잡도
- 사용 빈도

**플랜 비교:**

| 플랜 | 월 요금 | 특징 |
|-----|--------|------|
| **기본 (Pay as you go)** | ~$20 | 사용한 만큼 과금 |
| **Max Plan** | $100 | 무제한 사용 |

**비용 절감 팁:**
- `/compact`로 대화 압축하기
- `/clear`로 불필요한 맥락 제거
- 구체적으로 요청해서 재작업 줄이기

### Q5: 코딩을 전혀 모르는데 배울 수 있나요?

**A:** **네! 한국어로 요청하면 AI가 모든 코딩을 대신 해줍니다.**

**가능한 것:**
- 코드를 몰라도 원하는 결과물 생성 가능
- "이런 파일 만들어줘"만 하면 됨
- 코드를 이해할 필요 없음

**선택사항:**
1. **이해하지 않고 사용** — 결과물만 사용
2. **설명받기** — Claude에게 설명 요청
3. **배우기** — 시간을 들여 학습

```
예시:
> HTML 웹사이트를 만들어줄 수 있을까?
✓ (코드 생성)

> 이 코드가 뭐 하는 거야?
✓ (설명)

> 이걸 좀 더 배우고 싶어
✓ (학습 자료 제시)
```

### Q6: 윈도우, 맥 모두 사용 가능한가요?

**A:** **네, Windows와 Mac 모두 동일하게 사용 가능합니다.**

**설치 방법도 동일:**
- 둘 다 Node.js 설치
- 둘 다 `npm install` 명령어
- 둘 다 `claude` 명령어로 실행

**터미널이 다른 것뿐:**
- Windows = 명령 프롬프트 (cmd)
- Mac = 터미널

**기타 차이점:**
| 구분 | Windows | Mac |
|------|---------|-----|
| 실행 | 동일 | 동일 |
| 파일 경로 | C:\Users\... | /Users/... |
| 명령어 | cmd | 터미널 |

### Q7: 파일 수정할 때 원본이 사라지나요?

**A:** **기본적으로 원본은 보존되고 수정본으로 변경**됩니다.

```
상황:
1. main.py 파일이 있음
2. Claude가 수정 요청
3. 원본 main.py는 사라지고 수정된 내용으로 변경

안심할 점:
- 파일명은 그대로 유지
- 내용만 변경
- 자동으로 백업할 수 있음
```

**더 안전하게 하는 법:**
1. 수정 전에 파일 복사
```bash
> 작업 전에 main.py를 backup_main.py로 복사해줘
```

2. 복사본으로 작업
```bash
> backup_main.py를 수정해줄래?
```

### Q8: 클로드코드로 돈을 벌 수 있나요?

**A:** **네, 가능합니다!**

**활용 예시:**

| 사업 | 설명 |
|------|------|
| **프리랜싱** | 웹사이트 제작, 자동화 스크립트 개발 |
| **온라인 쇼핑몰** | 상품 소개 페이지 자동 생성 |
| **콘텐츠 제작** | 블로그 글, SNS 콘텐츠 대량 생성 |
| **데이터 처리** | 엑셀 데이터 정리 자동화 |

**사업 전략:**
1. 클로드코드로 빠르게 제작
2. 저렴하게 판매
3. 수익 창출

### Q9: 오래된 데이터를 처리할 수 있나요?

**A:** **네, 파일 형식에 관계없이 처리 가능합니다.**

**처리 가능한 데이터:**
- 텍스트 파일 (.txt)
- 엑셀 (.xlsx, .csv)
- 이미지 (.jpg, .png)
- 코드 파일 (모든 언어)
- 문서 (.pdf, .docx)

```
예시:
> 10년 전에 만든 엑셀 파일이 있어.
  이걸 읽어서 정렬해줄 수 있을까?
✓ 가능합니다
```

### Q10: 친구와 함께 사용할 수 있나요?

**A:** **각자 계정이 필요합니다.**

**상황별 설명:**
- 한 명의 Claude 계정 = 한 명이 사용
- 여러 명이 사용하려면 각각 API Key 필요
- 공유 계정은 가능하지만 권장 안 함

**협업 방법:**
1. 각자 계정 생성
2. 작업 파일 공유 (Dropbox, Google Drive)
3. Claude Code로 독립적으로 작업
