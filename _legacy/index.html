PART 2 · 설치 Step 1

Node.js 설치하기

클로드코드가 실행되려면 Node.js가 먼저 필요해요

🤔 Node.js가 뭔가요?
JavaScript 실행 환경입니다. 클로드코드가 동작하기 위해 필요한 기반 프로그램으로, 설치 후 특별히 쓸 일은 없습니다.
📥 설치 방법
1
nodejs.org 접속 → LTS 버전 다운로드 클릭
2
다운로드된 설치파일 실행 → Next 연타
3
설치 완료 후 아래 명령어로 확인
터미널 / 명령 프롬프트
$ node --version
v22.14.0  # 이렇게 나오면 성공!
⚠️ 터미널이 뭔가요?
명령어를 입력하는 검은 창입니다.

Windows: 검색창에 "cmd" 입력
Mac: Spotlight에서 "터미널" 검색
✅ 이미 설치됐는지 확인
$ node -v
# 버전 숫자가 나오면 이미 설치됨
# 오류가 나오면 새로 설치 필요
PART 2 · 설치 Step 2

Claude Code 설치하기

명령어 한 줄이면 설치 완료!

터미널을 열고 아래 명령어를 그대로 복사·붙여넣기 하세요.
터미널 — 설치 명령어
# 1단계: 클로드코드 설치
$ npm install -g @anthropic-ai/claude-code
# 2단계: 설치 확인
$ claude --version
1.x.x  # 버전이 나오면 성공!
npm은 Node.js 설치 시 자동으로 같이 설치됩니다.
"글로벌 설치(-g)"란 어느 폴더에서나 사용 가능하다는 뜻이에요.
⚠️ 오류 났을 때
🔴 permission denied 오류
# Mac/Linux — 앞에 sudo 붙이기
$ sudo npm install -g @anthropic-ai/claude-code
🟡 Windows 보안 오류
# PowerShell 관리자 권한으로 실행
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
💡 그래도 안 되면?
claude.ai에서 클로드에게 오류 메시지를 복사해서 물어보세요! AI가 직접 해결책을 알려줍니다.

Tip 2

영어 필요 없어요. 완전한 한국어로 대화해도 완벽히 이해합니다
Tip 3

PART 5 · 팁 & FAQ

자주 묻는 질문 FAQ

ChatGPT랑 다른 게 뭔가요?
ChatGPT는 대화만 합니다. 클로드코드는 실제로 내 컴퓨터 파일을 읽고 만들고 수정까지 합니다. 실행 능력이 핵심 차이입니다.
인터넷이 없으면 못 쓰나요?
네, 인터넷 연결이 필요합니다. AI 모델이 Anthropic 서버에서 동작하기 때문에 클라우드 연결이 필수입니다.
내 파일이 외부로 유출되나요?
대화 내용은 Anthropic 서버로 전송됩니다. 민감한 개인정보나 기업 기밀은 입력하지 않는 것이 좋습니다.
월 얼마나 나오나요?
사용량에 따라 다르지만 일반적으로 월 $5~20 수준입니다. Max Plan($100/월)은 무제한 사용 가능합니다.
코딩을 전혀 모르는데 배울 수 있나요?
네! 한국어로 요청하면 AI가 모든 코딩을 대신 해줍니다. 코드를 이해하지 않아도 원하는 결과물을 만들 수 있습니다.
윈도우, 맥 모두 사용 가능한가요?
네, Windows와 Mac 모두 동일하게 사용 가능합니다. 설치 방법도 동일합니다.
